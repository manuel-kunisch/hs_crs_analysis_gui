# Examples

This section will contain complete analysis examples built from real or simulated datasets.

The tutorials explain the GUI concepts. The examples should instead show full workflows from input data to final exported figures.

Planned examples:

- Reproducing Figure 1 for the paper.
- Synthetic quickstart data for testing installation.
- CARS/SRS label-free decomposition.
- SWIR or multispectral reflection unmixing.
- 4D fluorescence unmixing.
- Stitching and preprocessing workflow.

Each example should eventually include:

- Input data description.
- Required preset files.
- Expected analysis settings.
- Step-by-step GUI actions.
- Exported TIFF and CSV outputs.
- Placeholder or final figure panels.
